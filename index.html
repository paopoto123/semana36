errores en un algoritmo
errores de impresion: este error se da cuando escribimos mal el cout o no ponemos los signos como es
errores de peticion: esto se da cuando no se bien variable por ejemplo cuando no ponemos punto y coma
errores de modificacion : este error se da cuando no guardamos una variable o nombramos una que no existe
errores de declaracion : estos errores ocurren cuando estamos escribiendo algo incorrecto por ejemplo cuando no le ponemos comilla doble a una variable de string
